# isidoro-configuration (skeleton)

Arquivos de configuração e memória do Isidoro.

- `memory/`: arquivos de memória local (brain.db) — *NÃO* versionar dados sensíveis
- `state/`: JSONs de estado de runtime
- `AGENTS.md`, `SOUL.md`, `IDENTITY.md` etc. devem ser colocados aqui

Manter permissões restritas para `brain.db` (400).
