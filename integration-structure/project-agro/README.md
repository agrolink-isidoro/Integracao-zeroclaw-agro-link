# project-agro (skeleton)

Skeleton recomendado para o backend Agrolink (Django).

- `apps/`: módulos django (agricultura, estoque, fazendas, maquinas)
- `core/`: models comuns, middleware (tenant + JWT)
- `docker/`: compose e exemplos de deploy local

Preencher com `manage.py`, `requirements.txt` e demais artefatos quando implementar.
