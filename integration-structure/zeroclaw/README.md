# zeroclaw (skeleton)

Skeleton recomendado para o agente ZeroClaw (Node.js/Python adapter).

- `agents/`: definições de agentes e prompts
- `integrations/`: clientes REST para Agrolink (com JWT)
- `config.toml`: provider config example

Adicionar `integrations/agrolink_client.py` e `agents/isidoro/` conforme documentação.
